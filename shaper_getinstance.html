Shaper.getInstance()
Description
This static method returns an instance of one of the built in shape classes, ie Shaper.COSINE. See Shaper for a list of available shapes. This is a recommended method of getting access to a Shaper when shaping a large amount of data due to the fact that Shaper's static shape() method creates a new instance of the Shaper class on each call, creating a significant performance hit. Shapers with random elements initalize on construction, requiring this method over the static shape() method.
Example
import megamu.shapetween.*;

Shaper s = Shaper.getInstance( Shaper.COSINE );

beginShape();
for( float i=0; i<=1; i+= 0.05 )
  vertex( i*width, s.shape( i )*height );
endShape();
Syntax
getInstance(shape)
Parameters
shape
One of the built in shape classes, ie Shaper.COSINE
