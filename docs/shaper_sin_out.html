Shaper.SIN_OUT
Description
Sin out creates an ease-out using a sin function.
Shape
Shaper.shape(Shaper.SIN_OUT, 0.000); // → 0.000
Shaper.shape(Shaper.SIN_OUT, 0.125); // → 0.195
Shaper.shape(Shaper.SIN_OUT, 0.250); // → 0.383
Shaper.shape(Shaper.SIN_OUT, 0.375); // → 0.556
Shaper.shape(Shaper.SIN_OUT, 0.500); // → 0.707
Shaper.shape(Shaper.SIN_OUT, 0.625); // → 0.831
Shaper.shape(Shaper.SIN_OUT, 0.750); // → 0.924
Shaper.shape(Shaper.SIN_OUT, 0.875); // → 0.981
Shaper.shape(Shaper.SIN_OUT, 1.000); // → 1.000
Implementation
float shape( float in ){
  return sin( in*0.5*PI );
}
