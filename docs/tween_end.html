Tween::end()
Description
Ends the tween, setting the time to 1. If this tween is playing in reverse, this will set the time to 0.
Example
import megamu.shapetween.*;

Tween ani;
void setup(){
  ani = new Tween(this, 2);
  ani.repeat();
}

void draw(){
  background(255);
  ellipse(ani.time()*width, ani.position()*height, 4, 4);
}

void mousePressed(){
  ani.end();
}
