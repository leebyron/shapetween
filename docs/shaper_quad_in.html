Shaper.QUAD_IN
Description
Quadradic in creates an ease-in using a quadradic function.
Shape
Shaper.shape(Shaper.QUAD_IN, 0.000); // → 0.000
Shaper.shape(Shaper.QUAD_IN, 0.125); // → 0.016
Shaper.shape(Shaper.QUAD_IN, 0.250); // → 0.062
Shaper.shape(Shaper.QUAD_IN, 0.375); // → 0.141
Shaper.shape(Shaper.QUAD_IN, 0.500); // → 0.250
Shaper.shape(Shaper.QUAD_IN, 0.625); // → 0.391
Shaper.shape(Shaper.QUAD_IN, 0.750); // → 0.562
Shaper.shape(Shaper.QUAD_IN, 0.875); // → 0.766
Shaper.shape(Shaper.QUAD_IN, 1.000); // → 1.000
Implementation
float shape( float in ){
  return in*in;
}
