Shaper.SIN_IN
Description
Sin in creates an ease-in using a sin function.
Shape
Shaper.shape(Shaper.SIN_IN, 0.000); // → 0.000
Shaper.shape(Shaper.SIN_IN, 0.125); // → 0.019
Shaper.shape(Shaper.SIN_IN, 0.250); // → 0.076
Shaper.shape(Shaper.SIN_IN, 0.375); // → 0.169
Shaper.shape(Shaper.SIN_IN, 0.500); // → 0.293
Shaper.shape(Shaper.SIN_IN, 0.625); // → 0.444
Shaper.shape(Shaper.SIN_IN, 0.750); // → 0.617
Shaper.shape(Shaper.SIN_IN, 0.875); // → 0.805
Shaper.shape(Shaper.SIN_IN, 1.000); // → 1.000
Implementation
float shape( float in ){
  return ( 1 - cos( in*0.5*PI ) );
}
